# SteamChineseChecker

由[此帖](https://keylol.com/t503313-1-1)启发。

这是一个能在 Steam 商店页面显示游戏是否存在第三方汉化补丁的脚本。

数据来源是 Steam 鉴赏家：[华语汉化](https://store.steampowered.com/curator/31318556/)。

`data.json` 文件现在由 Github Actions 自动构建，每周拉取更新。
